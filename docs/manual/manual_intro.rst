***********************************************
Introduction
***********************************************

This document is meant to provide detailed information on the features of Vertiq modules. This manual is still being updated, not all features and 
configurations are covered yet. For more information and documentation, refer to the `Vertiq website <https://www.vertiq.co/>`_.

This manual is divided into sections based on individual features. The section for each feature contains details on which types of modules support that feature.
