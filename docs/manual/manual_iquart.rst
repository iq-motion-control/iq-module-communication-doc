***********************************************
IQUART
***********************************************

IQUART is a custom serial protocol supported by Vertiq modules. For more information on IQUART, refer to the :ref:`uart_messaging` and :ref:`getting_started_python_api` 
sections of the Vertiq documentation.

Module Support
===============
All Vertiq modules support IQUART, as summarized in the tables below.
